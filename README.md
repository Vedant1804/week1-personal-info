# Personal Information Manager

## Project Description
This is my first Python project! It stores and displays personal information while practicing f-strings and user input.

## Features
- Stores Name, Age, City, and University.
- Calculates age in months ($age \times 12$).
- Formats output with clean borders.

## How to Run
1. Run: `python personal_info.py`
2. Follow the prompts!
